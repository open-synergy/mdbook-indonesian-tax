PTKP Category

PTKP Category adalah fitur yang digunakan untuk mencatat dan mengelola konfigurasi data kategori Penghasilan Tidak Kena Pajak (PTKP).

  • Penjelasan PTKP Category
  • Membuat PTKP Category
  • Memodifikasi PTKP Category
  • Menghapus PTKP Category

Penjelasan PTKP Category

HEADER

Category

Nama Kategori.

Additional Note

Catatan tambahan.

Membuat PTKP Category

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Buka menu Taxform -> Configuration -> PPh 21 -> PTKP Category. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Klik tombol Create pada bagian atas-kiri form.

  1. Isi Category. Harus diisi.
  2. Isi Additional Note. Tidak harus diisi.
  3. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data PTKP Category akan terbuat.

Memodifikasi PTKP Category

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Buka menu Taxform -> Configuration -> PPh 21 -> PTKP Category. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data PTKP Category yang akan dimodifikasi. Abaikan jika data sudah dibuka.
  3. Klik tombol Edit pada bagian atas-kiri form.

  1. Isi dan sesuaikan Category jika dibutuhkan. Harus diisi.
  2. Isi dan sesuaikan Additional Note jika dibutuhkan. Harus diisi.
  3. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data PTKP Category akan berubah sesuai dengan perubahan yang dilakukan.

Menghapus PTKP Category

A. INPUT

  • User yang akan menghapus harus memiliki akses untuk menghapus PTKP Category.

B. LANGKAH KERJA

  1. Buka menu Taxform -> Configuration -> PPh 21 -> PTKP Category. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data PTKP Category yang akan dihapus. Abaikan jika data sudah dibuka.
  3. Klik tombol More pada bagian atas-tengah form.

  1. Klik tombol Delete pada dropdown yang muncul ketika tombol More diklik.

  1. Klik tombol Ok pada pop-up konfirmasi penghapusan yang muncul.

C. OUTPUT

  • Data PTKP Category akan terhapus.

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Klik label Add an Item pada bagian atas-kiri tabel Tarif PTKP

Pop-up Detail Tarif akan muncul.

  1. Pilih PTKP. Tidak harus diisi.
  2. Pilih PTKP Category. Harus diisi.
  3. Isi Tarif PTKP. Harus diisi.
  4. Klik tombol Save & Close pada bagian bawah-kiri pop-up Detail Tarif untuk menyimpan data. Klik tombol Save & New pada bagian bawah-kiri pop-up Detail Tarif untuk menyimpan data dan menambahkan data baru.

  1. Ulangi langkah ke-2 jika pada langkah ke-5 tombol Save & New yang dipilih.
  2. Lanjutkan langkah ke-5 instruksi kerja Membuat Tarif PTKP atau langkah ke-6 instruksi kerja Memodifikasi Tarif PTKP.

C. OUTPUT

(Tidak ada instruksi khusus)
